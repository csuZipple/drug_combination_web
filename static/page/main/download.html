Example Data Format:

idfidsourcedrug1namedrug2nameconc1conc2growthcellline
11VemurafenibVemurafenib0.0000.0000.630635764944276A-375PMC5248573

Column descriptions:

  • id: key
  • fid: foreign key
  • source: the origin data
  • drug1name: drug name
  • drug2name: drug name
  • conc1: drug concentration
  • conc2: drug concentration
  • growth: Combined drug sensitivity score for this cancer cell line, between 0-1
  • cellline: The name of the cancer cell line
